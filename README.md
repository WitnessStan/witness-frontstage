# iView-project

This project is build for Vue.js 2 + vue-router + webpack2 + iView 3, just install and run.

### 简介
```bush
基于iview的前台系统，使用axios与后端通信
webpack打包后部署到nginx
```

## Install
```bush
// install dependencies
npm install
```
## Run
### Development
```bush
// For the first time, run init to create index.html
npm run init
npm run dev
```
### Production(Build)
```bush
npm run build
```
